층화 추출법(Stratired)
범주 비율과 데이터 세트의 비율을 동일하게 나눠서 표본을 추출하는 기법
train_test_split 함수에 stratify 범주로 사용


분류분석 성능평가
4가지 지표
- 정확도 : 전체 중 올바르게 분류한 비율
(TP+TN)/(TP+FN+FP+TN) 
- 민감도(재현률) : 실제 전체 긍정 수 중 참 긍정의 비율
TP/(TP+FN)
- 특이도 : 실제 전체 부정수 중 참 부정의 비율
TN/(TN+FP)
- 정밀도 : 예측 전체 긍정 중 참 긍정의 비율
TP/(TP+FP)

ex)
TP FN
FP TN
	암	일반
암	10	0
일반	90	0
정확도 : 10/100 = 0.1
민감도 : 10/10 = 1
특이도 : 0/90 = 0
정밀도 : 10/90 = 0.11


트리
분류로 많이 쓰지만 회귀로도 쓰임
평균값을 기준으로 분할 기준을 정함
- Regression Tree 분할기준을 이용하여 각 영역의 평균값 예측
과대적합이 될 가능성이 매우 높음
=> 가지치기로 완화(분류기준을 제거)


앙상블 Ensemble
여러 머신 러닝 모델을 결합해 더 강력한 모델을 만드는 기법
- Voting 보팅
서로다른 알고리즘을 결합

- Bagging 배깅
같은 알고리즘을 결합, 서로다른 훈련 데이터로 학습
=> 중복 추출을 허용
- n_estimators
생성할 트리의 개수, 클수록 좋지만 훈련 시간과 메모리 부담
- max_features
선택할 무작위 특성의 개수, 핵심 변수 기본값 권장
값이크면 : 트리들이 비슷, 작으면 : 트리들이 달라지고 깊어짐

- Boosting 부스팅 => 가장 강력한 방법
여러 모델이 순차적으로 학습, 이전 모델에서 발생한 오류에 가중치를 높게 주어 학습
 Adaptive 과소 적합 된 훈련 가중치를 높여 새로운 모델 학습
 Gradient 비용함수의 기울기로 오답 가중치를 부여 (기울기가 높을수록 가중치상승)
